Práctica 4: 1_Attiny85 Digital I/O

Emmanuel Angeles Barreto
Ing. en Comunicaciones y Eletrónica
Fecha: 01/09/2024
Arduino Electrónica C++ Attiny

- Resumen -

Esta práctica permitió familiarizarse con el funcionamiento y la programación del Attiny85, sentando las bases para proyectos más avanzados de control digital.

Se exploraron las capacidades de entrada y salida digital del microcontrolador Attiny85. Se configuraron pines como entradas y salidas para realizar tareas básicas de control y monitoreo de dispositivos. Se implementó un programa que permite encender y apagar un LED mediante la lectura de un pulsador.

- Introducción -

A través de la configuración de pines como entradas y salidas, se desarrollarán habilidades fundamentales en la programación y el control de dispositivos, estableciendo una base sólida para futuros proyectos de automatización y control.

El objetivo principal de esta práctica es familiarizarse con las funcionalidades de entrada y salida digital del Attiny85.

- Materiales -

- Arduino Uno

- Attiny

- Protoboard

- Cables

- LED´´S

- Boton

- Desarrollo -

Paso 1: Conectar el Potenciómetro:

- Asegurando que el microcontrolador estuviera correctamente conectado a la placa de desarrollo y que el software de programación estuviera listo para utilizar.

Paso 2: Conexiones de cada circuito a realizar:

- Se diseñó un circuito básico que incluía un LED y un pulsador. El LED se conectó a uno de los pines de salida del Attiny85, mientras que el pulsador se conectó a un pin de entrada, permitiendo al microcontrolador leer su estado.

Paso 3: Programación

- Se escribió un código en C para el Attiny85 utilizando el entorno de desarrollo adecuado

- Diseño -

El diseño del circuito es sencillo y se puede implementar en una protoboard:

El potenciómetro tiene tres pines: uno conectado a Vcc (5V), otro a GND y el pin del centro conectado a un pin analógico de Arduino (A0).

El servomotor tiene tres cables: el cable de señal (amarillo) conectado a un pin digital del Arduino con PWM (por ejemplo, el pin 9), el cable rojo conectado a 5V y el cable negro a GND.

Arduino puede alimentarse directamente por USB o mediante una fuente externa si es necesario.

- Construcción: -

Los circuitos se construyen sobre un protoboard, siguiendo el esquema de conexión. Es importante respetar la polaridad de los LEDs y las conexiones de los botones para evitar cortocircuitos.

- Resultados y Simulación -

El servomotor debe moverse de acuerdo con la posición del potenciómetro. Al girar el potenciómetro, se debería observar que el ángulo del servomotor varía entre 0 y 180 grados. Este resultado demuestra cómo el valor analógico del potenciómetro se convierte en una señal PWM que controla la posición angular del servomotor.

- Conclusiones -

La práctica ha demostrado la capacidad de Arduino para controlar un servomotor utilizando una entrada analógica.

Se ha logrado convertir el valor analógico del potenciómetro en un ángulo de rotación del servomotor, con lo que se cumple el objetivo de controlar la posición del motor de manera precisa y eficiente.
